# Workbook Templates

This folder contains template workbooks intended to be committed to GitHub.

- `general/`: standard operating-company template for non-FIG companies.
- `fig/`: FIG-oriented templates. The JEF workbook is the repo's current
  configured financial-services / broker-dealer reference template.

Use the general template for ordinary operating companies. Use a FIG template
when the company requires financial-services statement structure rather than
standard revenue, cost of revenue, and operating-company balance-sheet rows.
